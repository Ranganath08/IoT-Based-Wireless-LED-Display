# IoT-Based-Wireless-LED-Display

Developed an IoT-enabled wireless LED display using an ESP8266 (NodeMCU) and P10 LED matrix panels (HUB12 interface). The system hosts a built-in web server that allows users to update display messages in real time over Wi-Fi without requiring a wired connection. Implemented support for dynamic scrolling and static text display, enabling remote content management through any web browser. The project demonstrates practical applications of embedded systems, IoT networking, and real-time display control, making it suitable for use in educational institutions, public information boards, shops, and event announcements.

Technologies Used: ESP8266, Arduino IDE, Embedded C/C++, Wi-Fi, HTTP Web Server, P10 LED Matrix (HUB12), DMDESP Library.
